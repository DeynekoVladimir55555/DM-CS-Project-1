# DM-CS-Project-1
Create a computer algebra system. 

[Ссылка на таблицу с модулями](https://docs.google.com/document/d/1TNSF2cwcd-7z0w6qAfidsigiQliIXhzCgGgf0g6X1Yw/edit?tab=t.0)

Для тех, кто делает модули в "первую волну":
  1. Создаете свою ветку, в названии указать номер задачи из первого столбца таблицы и свое имя. (например DeinekoV-N-1)
  2. Ваши модули неодходимо написать внутри самих обьектов. Открываете соответствующий файл из src/DataClasses с нужным вам классом ( если натуральные числа то NaturalNumber.py и т.д.)
  3. Создаете в классе функцию с названием модуля из таблицы, но БЕЗ капса ( COM_NN_D -> com_nn_d )
  4. Реализуете логику и т.п. Оформляете в соответствии к требованиям. (комментарий кто делал, описание функции, возможные комментарии сложных частей кода)
  5. При создании коммита в комментарии к нему в 2-3 словах описать, что за задача. (например, "сложение натуральных чисел")
  6. Отправляете и делаете пулл-реквест.
  7. Ждете проверки. В случае требования исправлений, будет оставлен комментарий и будете тегнуты в группе.
